OWASP Top 10 Security Risks

A01:2021

Broken Access Control

Restrictions on what authenticated users are allowed to do are not properly enforced.

// Vulnerability: Direct object reference
http://app.com/user/123/profile
Prevention: Implement proper authorization checks, use indirect object references
A02:2021

Cryptographic Failures

Failure to protect sensitive data through encryption and proper key management.

// Vulnerability: Plaintext passwords
INSERT INTO users VALUES ('admin', 'password123')
Prevention: Use strong encryption, secure key management, TLS everywhere
A03:2021

Injection

Untrusted data is sent to an interpreter as part of a command or query.

// SQL Injection example
SELECT * FROM users WHERE username = '' OR '1'='1'
Prevention: Use parameterized queries, input validation, ORM frameworks
A04:2021

Insecure Design

Security flaws in the architecture and design of the application.

// Design flaw: No rate limiting
// Allows unlimited login attempts
Prevention: Threat modeling, secure design patterns, security requirements
A05:2021

Security Misconfiguration

Insecure configurations in any part of the application stack.

// Vulnerability: Default credentials
// admin/admin still enabled in production
Prevention: Hardened configurations, automated scanning, minimal platforms
A06:2021

Vulnerable Components

Using dependencies with known vulnerabilities.

// Outdated library with CVE
log4j 1.2.17 (CVE-2021-44228)
Prevention: Software composition analysis, patch management, vulnerability scanning

Secure Coding Principles

Input Validation

Validate all inputs from untrusted sources. Use allowlists rather than denylists.

Authentication & Authorization

Implement strong authentication and proper authorization checks for all requests.

Cryptography

Use strong, standard cryptographic algorithms and proper key management.

Error Handling

Handle errors gracefully without exposing sensitive information.

Secure Database Access

Use parameterized queries and stored procedures to prevent SQL injection.

Session Management

Implement secure session management with proper timeout and rotation.
