Infrastructure as Code (Terraform)

Secure AWS infrastructure provisioning with automated security controls

# AWS Security Group with least privilege
resource "aws_security_group" "web_sg" {
  name_prefix = "web-"
  vpc_id      = var.vpc_id

  ingress {
    from_port   = 443
    to_port     = 443
    protocol    = "tcp"
    cidr_blocks = ["0.0.0.0/0"]
  }

  egress {
    from_port   = 0
    to_port     = 0
    protocol    = "-1"
    cidr_blocks = ["0.0.0.0/0"]
  }

  tags = {
    Environment = var.environment
    SecurityLevel = "high"
  }
}

CI/CD Security Pipeline

GitHub Actions workflow with comprehensive security scanning

name: DevSecOps Pipeline
on: [push, pull_request]

jobs:
  security-scan:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run Trivy vulnerability scanner
        uses: aquasecurity/trivy-action@master
        with:
          scan-type: 'fs'
          scan-ref: '.'
          format: 'sarif'
          output: 'trivy-results.sarif'
      - name: Upload Trivy scan results
        uses: github/codeql-action/upload-sarif@v2
        with:
          sarif_file: 'trivy-results.sarif'

Azure Security Center Automation

Automated security recommendations and remediation

# PowerShell script for automated remediation
param(
    [string]$ResourceGroupName,
    [string]$SubscriptionId
)

# Connect to Azure
Connect-AzAccount

# Get security recommendations
$recommendations = Get-AzSecurityRecommendation

foreach ($rec in $recommendations) {
    if ($rec.State -eq "Active") {
        # Apply remediation
        Invoke-AzSecurityRemediation -Name $rec.Name
    }
}

GCP DevSecOps Automation

Security Command Center Integration

Automated security findings and response using SCC API

# Python script for SCC automation
import google.cloud.securitycenter as securitycenter

def create_finding(project_id, source_id, finding_id):
    client = securitycenter.SecurityCenterClient()
    
    finding = {
        "name": f"organizations/{org_id}/sources/{source_id}/findings/{finding_id}",
        "state": "ACTIVE",
        "resource_name": f"//compute.googleapis.com/projects/{project_id}/zones/us-central1-a/instances/example-instance",
        "category": "PII_DATA_ACCESS",
        "external_uri": "https://console.cloud.google.com/iam-admin/iam",
        "severity": "HIGH",
        "event_time": {"seconds": 1556753400},
    }
    
    response = client.create_finding(
        request={"parent": f"organizations/{org_id}/sources/{source_id}", "finding": finding}
    )
    return response
